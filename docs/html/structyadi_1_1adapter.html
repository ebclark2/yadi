Detailed Description

template<typename FT, typename OT = ptr_type_t<meta::derive_base_type_t<FT>>>
struct yadi::adapter< FT, OT >

Provides a layer above factory<BT>::create. This allows types such as templated containers to be created without registering initializers for each element type.

Template Parameters
FTThe type the base type will be derived from via derived_base_type_t<FT>.
OTThe desired output type.
